🚀 คู่มือการติดตั้งและรันโปรเจกต์ SafeScan สำหรับเพื่อนร่วมทีม 🚀
============================================================

ให้ทำตามขั้นตอนด้านล่างนี้เรียงตามลำดับเพื่อรันระบบในเครื่องตัวเองนะ:

------------------------------------------------------------
🛠️ สเต็ปที่ 1: เตรียมโปรแกรมในเครื่อง (ถ้ามีแล้วข้ามได้เลย)
------------------------------------------------------------
1. ติดตั้ง Node.js (แนะนำเวอร์ชัน LTS)
2. ติดตั้ง Git (สำหรับ Windows ให้โหลด Git for Windows)
3. ติดตั้ง MySQL Workbench และเปิดบริการ MySQL Server ให้เรียบร้อย

------------------------------------------------------------
📥 สเต็ปที่ 2: ดึงโค้ดลงเครื่อง (Clone Project)
------------------------------------------------------------
1. สร้างโฟลเดอร์ว่าง ๆ ในเครื่องคอมพิวเตอร์ของนาย
2. เปิด Command Prompt / Terminal ในโฟลเดอร์นั้น
3. พิมพ์คำสั่งนี้แล้วกด Enter เพื่อดึงโค้ดลงมา:
   git clone https://github.com/EKKA843/SafeScan-Project.git

4. พอโหลดเสร็จ ให้พิมพ์คำสั่งเดินเข้าโฟลเดอร์โปรเจกต์:
   cd SafeScan-Project

------------------------------------------------------------
📦 สเต็ปที่ 3: ติดตั้ง Library (เนื่องจากไม่ได้เอา node_modules ขึ้นไป)
------------------------------------------------------------
เนื่องจากโฟลเดอร์ node_modules มีขนาดใหญ่ เราจึงต้องสั่งติดตั้งใหม่ในเครื่องตัวเอง โดยให้เปิด Terminal แยกทำ 2 ส่วนดังนี้:

▶️ ส่วนที่ 3.1: ฝั่งหลังบ้าน (Backend)
1. เปิด Terminal เดินเข้าโฟลเดอร์ backend (ปรับชื่อตามโฟลเดอร์จริง)
   cd backend
2. พิมพ์คำสั่งติดตั้ง:
   npm install

▶️ ส่วนที่ 3.2: ฝั่งหน้าบ้าน (Frontend)
1. เปิด Terminal อีกหน้าต่าง เดินเข้าโฟลเดอร์ frontend
   cd frontend
2. พิมพ์คำสั่งติดตั้ง:
   npm install

------------------------------------------------------------
🗄️ สเต็ปที่ 4: ตั้งค่าฐานข้อมูล (MySQL Workbench)
------------------------------------------------------------
1. เปิดโปรแกรม MySQL Workbench ในเครื่องของนาย
2. มองหาแถบ Navigator ด้านซ้าย ในหมวด Management ให้คลิกเลือก "Data Import/Restore"
3. ในหัวข้อ Import Options ให้ติ๊กเลือก "Import from Self-Contained File" 
4. กดปุ่ม `...` เพื่อไปเลือกไฟล์ฐานข้อมูล `.sql` (ไฟล์ที่ Beam ส่งให้ในแชทกลุ่ม)
5. ตรงช่อง "Default Target Schema" ให้กดปุ่ม "New..." เพื่อสร้าง Database ใหม่ในเครื่อง *โดยตั้งชื่อให้ตรงกับฐานข้อมูลที่เขียนไว้ในไฟล์คอนฟิกหลังบ้าน*
6. คลิกที่แท็บ "Import Progress" (อยู่ด้านบนข้างๆ แท็บ Object Selection) แล้วกดปุ่ม "Start Import" ที่มุมขวาล่างสุด

⚠️ สำคัญมากก่อนรัน: 
ให้เข้าไปเปิดไฟล์ `src/config/db.js` ในโฟลเดอร์ backend แล้วแก้ไขค่า `user` และ `password` ของ MySQL ให้ตรงกับรหัสผ่านที่นายใช้เข้า MySQL Workbench ในเครื่องตัวเองด้วยนะ!

------------------------------------------------------------
🚀 สเต็ปที่ 5: วิธีสั่งรันโปรเจกต์ (Start Project)
------------------------------------------------------------
เวลาจะเปิดใช้งานหรือพรีเซนต์งาน ให้เปิด Terminal รันขนานกัน 2 ตัวแบบนี้:

1. เปิด Terminal ตัวที่ 1 (โฟลเดอร์ backend):
   npm run dev
   (หรือ npm start ตามที่เซ็ตไว้ เพื่อรัน Server API หลังบ้าน)

2. เปิด Terminal ตัวที่ 2 (โฟลเดอร์ frontend):
   npm run dev
   (เพื่อรันหน้าเว็บหน้าบ้าน จากนั้นเปิดเว็บบราวเซอร์รันตามลิงก์ที่ระบบแจ้งได้เลย!)

------------------------------------------------------------
✍️ สเต็ปที่ 6: วิธีส่งงานคืนขึ้น GitHub เมื่อเขียนโค้ดเสร็จ
------------------------------------------------------------
เมื่อนายพิมพ์แก้ไขโค้ดพาร์ทของตัวเองเสร็จแล้ว และต้องการอัปเดตไฟล์กลับขึ้นคลาวด์ ให้พิมพ์ 3 คำสั่งนี้ในโฟลเดอร์หลักของโปรเจกต์:
1. git add .
2. git commit -m "feat: อัปเดตโค้ดหน้าจอพาร์ทของฉัน"
3. git push origin main
